# Flask Portfolio App

Production Python web application deployed on AWS EC2.

## Architecture
Browser → Nginx (port 80) → Gunicorn → Flask (port 5000)

## Tech Stack
- Python 3 / Flask
- Gunicorn (production WSGI server)
- Nginx (reverse proxy)
- AWS EC2 (Ubuntu 24.04)
- Systemd (service management)
